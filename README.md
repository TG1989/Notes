# React Note-taking App 📝

Welcome to the React Note-taking App! This simple yet powerful application allows you to manage your notes effortlessly. Built with React and React Router, it provides a seamless user experience for creating, editing, and viewing notes.

![my-notes](https://github.com/TG1989/Notes/assets/115742987/aa6dbcd3-fc64-4d20-bdb9-696c5c58ed57)


## Features 🚀

- **Notes Page:** Explore and manage your notes with ease.
- **Create Note:** Intuitive interface for adding new notes.
- **Edit Note:** Modify existing notes on-the-go.

## Tech Stack 💻

- React
- React Router
- useState and useEffect hooks for state management
- LocalStorage for data persistence
